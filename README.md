Setup for dev testing:

1. install rvm
2. `rvm install 1.8.7-r147`  # This site isn't *that* specific to ruby version. This is the just the ruby ver I'm using.
3. rvm --default 1.8.7-r147
4. gem install jekyll
5. gem install rdiscount
